Experimental Robotics
(CSCI 1108) Final Project:
Temple Run
 This project tested my abilities of working with a team, and finding an original solution given a difficult problem with no obvious solution. This project combined every aspect of experimental robotics to design code for a THYMIO II robot to navigate an unknown course, given various barcodes, and stop on the finish area.

Sprite The Game

 Sprite the Game is an intense FPS shooter where you play as a can of soda. This game is a work in progress, but will contain dynamic and elaborate health and movement mechanics. Sprite the Game shows the implementation of 3D player movement and graphics, and implementation of visual sprites (no pun intended) with the use of Unity.

Racing Galactic Asteroid Simulator

 Pilot a spaceship in this action-packed, space racing simulator. Racing Galactic is in its very early stages of development, but is planned to produce high-octane levels of racing, with Mario Kart style enjoyment and replayability. Racing Galactic will display its intricate asteroid field system, and 3D spaceship movement mechanics, developed with Unity.
